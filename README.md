# 🚦 Traffic Accident Analysis & Prediction System

## 📌 Project Description
This project focuses on analyzing traffic accident data to identify patterns, causes, and risk factors, and to build a simple machine learning model for predicting accident severity. The goal is to support better decision-making for road safety and traffic management.

The system uses data preprocessing, exploratory data analysis (EDA), and visualization techniques to extract meaningful insights from the dataset. Additionally, a basic machine learning model is implemented to make predictions based on key features such as location, weather conditions, and time.

## 🎯 Objectives
- Analyze traffic accident data to identify trends and patterns  
- Visualize accident hotspots and contributing factors  
- Build a simple and meaningful ML model for prediction  
- Provide insights for improving road safety  

## 🛠️ Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook  

## 📊 Features
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of accident trends  
- Feature selection  
- Machine Learning model (classification/regression)  
- Performance evaluation  

## 🤖 Machine Learning Model
A simple and effective model (such as Logistic Regression or Decision Tree) is used to:
- Predict accident severity  
- Analyze impact of different features  

## 📂 Dataset
The dataset contains traffic accident records including:
- Date and Time  
- Location  
- Weather conditions  
- Road conditions  
- Accident severity  

## 📈 Results
- Identified key factors contributing to accidents  
- Visualized high-risk zones and time periods  
- Achieved meaningful prediction accuracy using a basic ML model
